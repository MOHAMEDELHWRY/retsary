Place a Unicode Arabic TTF font here for PDF exports.

Recommended:
- Amiri-Regular.ttf (https://github.com/aliftype/amiri/releases)
- NotoNaskhArabic-Regular.ttf (https://github.com/googlefonts/noto-fonts)

Expected filename:
- Amiri-Regular.ttf

The app will try this local path first: /fonts/Amiri-Regular.ttf
If not present, it will fall back to CDN.
